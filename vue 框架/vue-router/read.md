# src 是项目开发的地方
assets 资源 放图片和css
componets组件
app.vue vue文件，写vue代码
到main.js里面去处理vue代码
index.html有一个id为app的空容器

现在我们就是把vue代码放在app.vue文件中

package.json 是整个项目的说明书

npm install默认安装项目说明书里面的依赖
、vite相当于live server

参照说明书
nem run dev

vite将整个项目中的文件打包，展现它们整合起来的效果

vite 有代码变更，自动刷新
vite叫构建工具
